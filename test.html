Brave 瀏覽器綁定 Uphold 交易所

前言

使用 Brave 瀏覽器的朋友應
    
    
    
    
    該已經習慣了，瀏覽網頁無廣告的體驗了吧，當然它的強大的安全性和隱私性，讓我們在上網時，更加安心，其實，像我還常常透過觀看批准的廣告來賺取 BAT 代幣，雖然一個月從廣告中賺到的 BAT
    代幣不多，而且市值偏低，但有總
    
    
    
    
    比沒有好，不是嗎？Basic Attention Token，又稱 BAT，是一種區塊鏈數位廣告平台的代幣，目的在於公平地獎勵用戶的注意力，同時也讓廣告主的廣告花費得到更好的回報。Brave
    目前支援的虛擬加密貨幣交易平台有兩個，分別為 Uphold 和 Gemini 交易所，Gemini 就是今年年初因為 FTX 交易所倒閉，而引起連鎖反應，可能也會倒閉的雙子星交易所。

    因為 Gemini 新聞比較多，監
    
    
    
    
    
    
    管可能會比較嚴格，所以，我選
    
    
    
    
    擇的是相對而言，較友善的 Uphold 交易所，在 FTX 事件過後，許多交易所開始展示他們的儲備證明，Uphold 以 100%
    的儲備金證明它的安全控管，不過，它可以交易的資產種類只有 200 多種，且對不同種類的資產收取不同種類的點差，交易價值低於 100 美元的交易均需支付 0.99 美元的小額交易費，重點是，Brave 瀏覽器可以綁定 Uphold
    交易所，把賺到的 BAT 代幣轉到 Uphold 上進行操作。

以下為綁定步驟

  1. 在Brave 瀏覽器右邊，點選 setting。
  2. 在打開的 setting 瀏覽頁中，點選 Brave Rewards，點選 Start Using Brave Rewards。
  3. 選擇國家為台灣 (Taiwan)。
  4. 完成後，再回到 Brave Rewards ，選擇 Connect Account。
  5. 我是選擇 uphold，之後便是一連串 uphold 交易所的登入驗證機制，我就省略細節。
完成綁定。
